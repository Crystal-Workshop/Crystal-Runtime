# Crystal Runtime
### All-In-One C++ runtime for games, using the .crg file format

## Setup Steps
Step 1:
[Install CMake](https://cmake.org/download/)
Step 2: 
Open Powershell at your project directory
Step 3:
Run ``cmake -G "Visual Studio 17 2022"``
Step 4:
Run ``cmake --build . --config Release``
Step 5:
Open a .crg file
